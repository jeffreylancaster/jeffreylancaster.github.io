• Game of Thrones

    This project started it all. I was curious whether I could visualize the various narrative threads woven throughout the Game of Thrones HBO series; to do that, I had to create a custom dataset with episode and scene metadata, character data, and more.

  • Stranger Things

    I was curious whether the Game of Thrones visualizations would work for another show, so I created a new dataset in the same format to extend the work to Stranger Things.

  • Marvel

    Marvel has done an excellent job weaving narrative threads throughout an extended series of films and shows, so I was curious how that might be visualized. This also led me to having a play with the Marvel Developers API.

  • Man in the High Castle

    When I heard that Amazon Prime includes data in a show about that show, I was curious to extract it and build upon the Game of Thrones and Stranger Things visualizations above, perhaps without needing a custom dataset.

  • Transit

    I've always loved transit maps; this is a playground for ideas of re-visualizing those maps using data and physics.
